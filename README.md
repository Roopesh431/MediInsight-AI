# MediInsight AI

An AI-powered platform for intelligent medical report and hospital bill analysis.

## Features (Planned)

- PDF Upload
- OCR for scanned documents
- Medical report parsing
- Hospital bill analysis
- AI-generated report summaries
- Retrieval-Augmented Generation (RAG)
- Interactive dashboard
- Secure user authentication

## Tech Stack

### Backend
- Python
- FastAPI

### Frontend
- React
- Tailwind CSS

### AI
- Hugging Face
- PaddleOCR
- SentenceTransformers

### Database
- SQLite
- PostgreSQL
